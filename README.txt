----------------
Pomodoro Desktop
----------------

Pomodoro desktop is a simple utility to manage a kitchen-like timer for your coding activities on your Mac.
You can find more informations on Pomodoro Technique on http://www.pomodorotechnique.com/
Updates, source code, new releases and fixes on http://pomodoro.ugolandini.com
 
-------
Manual
-------
Just start the pomodoro!
In Interrupt notifications text, you can use $secs placeholder to indicate the interrupt duration. 
In "Every" notification text, you can use $mins, $passed and $time placeholders to indicate respectively interval time, total time passed and how much time is left.
You need to install Growl Framework separately if you want growl notifications.
To reset default values, just type this command in your home directory:
rm com.ugolandini.pomodoro.plist
